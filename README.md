## Project Description

A personal portfolio website built with HTML, CSS, and JavaScript. It showcases my academic projects, work experience, and skills as a Software Engineering student at KFUPM. The site was developed across three assignments, with each iteration adding more functionality. Assignment 3 focused on advanced features including external API integration, complex filtering and sorting logic, state management, and performance improvements.

**Features include:**
- GitHub repositories fetched live from the GitHub API
- Inspirational quotes fetched from an external quotes API
- Project filtering by category and skill level, with sorting
- Login/logout simulation using sessionStorage
- Dark/light theme toggle persisted in localStorage
- Personalised greeting saved in localStorage
- Site timer counting time spent on the page
- Contact form with multi-step validation
- Responsive design for mobile and desktop

## Setup Instructions

No build tools or dependencies are required. The project runs as plain HTML/CSS/JS.

**To run locally:**

1. Clone the repository:
   ```bash
   git clone https://github.com/MohammedAlroomi/202250760-MohammedAlroomi-assignment3.git
   ```

2. Navigate into the project folder:
   ```bash
   cd 202250760-MohammedAlroomi-assignment3
   ```

3. Open `index.html` in your browser. You can either:
   - Double-click the file directly, or
   - Use a local server for best results (e.g. VS Code **Live Server** extension)

> **Note:** The GitHub API and Quotes API require an internet connection to load. The site works offline but those sections will show their error fallback messages.
